# Project Title

Team Directory App

## Overview

Team Directory App allows users to view and add team members with their names, roles, emails, and bios. It also allows users to delete members in Member Profile page.

## Local Setup Steps

Before starting the client and the mock server, please  run "npm i" or "npm install" to install all necessary dependencies.
- Edit .env.sample to access API URL
- After installation, run "npm run dev" to start the application
- In a separate terminal, run "npm run server" to start the mock server (data will be saved in the db.json file in the root directory)
- In a separate terminal, run "npm test" to execute test cases
